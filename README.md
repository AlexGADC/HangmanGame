Hangman is a small C# WPF application where you can play the classic hangman game.

Features:
- save/open game support: you can save a game and play it later
- words category: the game contains a list of categories for the words
- profile pictures: when a user logs into an account he can choose one of the profile pictures
- captivating hangman images: when you start making mistakes you can see the concern on the hangman face, just a little funny thing to enhance the player experience
